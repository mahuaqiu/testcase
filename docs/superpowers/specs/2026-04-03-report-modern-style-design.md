# 报告样式现代化设计

## 概述

将 HTML 测试报告从当前样式升级为清新活力的现代化风格。

## 设计决策

| 选项 | 选择 |
|------|------|
| 整体风格 | 清新活力 - 明亮配色，状态色块 |
| 配色方案 | 绿色系（成功）/ 红色系（失败） |
| 日志布局 | 卡片分层 - 左侧图标+信息，耗时紧跟标题 |
| 子步骤样式 | 缩进卡片 - 白色背景小卡片 |

## 配色规范

### 成功状态
- 主色: `#22c55e` (green-500)
- 浅背景: `#f0fdf4` (green-50)
- 徽章背景: `#dcfce7` (green-100)
- 文字: `#166534` (green-800)

### 失败状态
- 主色: `#ef4444` (red-500)
- 浅背景: `#fef2f2` (red-50)
- 徽章背景: `#fee2e2` (red-100)
- 文字: `#dc2626` (red-600)

### 中性色
- 标题文字: `#1f2937` (gray-800)
- 正文文字: `#374151` (gray-700)
- 次要文字: `#6b7280` (gray-500)
- 边框: `#e5e7eb` (gray-200)

## 布局结构

### 报告头部
- 渐变背景：`linear-gradient(135deg, #f0fdf4 0%, #dcfce7 100%)`
- 测试名称 + 描述
- 状态徽章（圆角胶囊，带阴影）
- 总耗时 + 时间戳

### 日志卡片
```
┌─────────────────────────────────────────────┐
│ ✓  LoginAW.do_login()  7.4s                │
│    userA · 麻嘉洋 · 22:54:08               │
├─────────────────────────────────────────────┤
│  ┌─────────────────────────────────────┐    │
│  │ ✓  ocr_wait("邮箱/帐号")      1.6s │    │
│  └─────────────────────────────────────┘    │
│  ┌─────────────────────────────────────┐    │
│  │ ✓  ocr_click("登录")         813ms │    │
│  └─────────────────────────────────────┘    │

```

### 失败卡片
- 红色渐变头部
- 失败步骤红色边框高亮
- 展示错误信息块

## 改动文件

| 文件 | 改动内容 |
|------|----------|
| `common/report_generator.py` | 重构 CSS 样式和 HTML 结构 |

## 交互保持

- 点击卡片展开/收起子步骤
- 失败卡片默认展开
- 点击子步骤展开详情
